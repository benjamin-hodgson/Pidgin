Examples

This example shows how to use mutual recursion to create a parser equivalent to Many()

// many is equivalent to String("foo").Separated(Char(' '))
var many = Rec(self =>
    String("foo").Then(
        Char(' ').Then(self).Optional(),
        (x, y) => x + y.GetValueOrDefault("")
    )
);
